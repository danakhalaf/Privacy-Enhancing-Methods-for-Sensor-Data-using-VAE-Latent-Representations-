# Privacy Enhancing Methods for Sensor Data using Variational Autoencoder Latent Representations

This repository contains the code developed for my Master of Professional Engineering thesis at the University of Western Australia.

The project re-implements and evaluates a variational autoencoder (VAE) based latent manipulation framework for privacy-preserving sensor data analysis using the MotionSense dataset. The goal is to investigate whether latent representations can be manipulated to reduce inference of sensitive attributes (gender) while preserving the utility of human activity recognition.

The repository includes code for:

* Data preprocessing and window generation
* Baseline activity and gender classifiers
* VAE and autoencoder training
* Deterministic and probabilistic latent mean-swap transformations
* Privacy–utility evaluation
* PCA and t-SNE latent space analysis
